# WanderLust – Tourism Management Platform 🌍

A full-stack tourism management web application inspired by travel booking platforms, built using Node.js, Express.js, MongoDB, and EJS. The platform allows users to create, manage, and explore travel stay listings through a dynamic and responsive interface.

⭐ Currently improving the project by adding authentication, image uploads, and deployment support.

📌 This project focuses on backend development, database integration, and server-side rendering concepts.

> 🚧 This project is currently under active development.

---

# Features

- Create new travel listings
- View all travel listings
- Edit existing listings
- Delete listings
- RESTful routing implementation
- Dynamic server-side rendering using EJS
- MongoDB database integration
- MVC-based project structure
- Responsive UI using Bootstrap

# Installation & Setup

1. Clone the repository

```bash
git clone <your-repository-link>
```

2. Navigate to the project folder

```bash
cd wanderlust-tourism-platform
```

3. Install dependencies

```bash
npm install
```

4. Start MongoDB locally

5. Run the application

```bash
node app.js
```

6. Open in browser

```bash
http://localhost:8080
```
